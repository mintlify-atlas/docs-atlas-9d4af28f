- Creates two entries: one `input`, one `output`

</ParamField>

<ParamField path="parent" type="tk.Widget">
  Notebook tab frame
</ParamField>

**Widgets created on `app`:**

| Widget | Type | Purpose |
|--------|------|----------|
| `app.gre_tunnel_id` | `tk.Entry` | Tunnel interface number (default: 0) |
| `app.gre_local_src` | `tk.Entry` | Local source IP (physical interface) |
| `app.gre_remote_dst` | `tk.Entry` | Remote peer IP (destination) |
| `app.gre_tunnel_ip` | `tk.Entry` | Tunnel IP (local end) |
| `app.gre_tunnel_mask` | `tk.Entry` | Tunnel subnet mask |
| `app.gre_isakmp_key` | `tk.Entry` | Pre-shared key (masked) |
| `app.gre_isakmp_peer` | `tk.Entry` | IPsec peer IP |
| `app.gre_map_name` | `tk.Entry` | Crypto map name (default: "CMAP_GRE") |
| `app.gre_map_seq` | `tk.Entry` | Crypto map sequence (default: 10) |
| `app.gre_bind_iface` | `tk.Entry` | Physical interface for crypto map |
| `app.gre_list` | `tk.Listbox` | Configured tunnels |
| `app.gre_tunnels` | `list[dict]` | In-memory tunnel configs |

**GRE over IPsec Architecture:**

```
Physical:  SW [Gi0/X] ← GRE in IPsec → [Gi0/X] Peer
Logical:   Tunnel0 (10.x.x.1) ←────────→ Tunnel0 (10.x.x.2)
```

**IKEv1 Phase 1 (ISAKMP):**
- Encryption: AES-256
- Hash: SHA-1
- Auth: Pre-shared key
- DH Group: 2 (1024-bit) — max supported by IOSvL2 15.2
- Lifetime: 86400 sec (24 hours)

**IPsec Phase 2 (ESP):**
- Transform-set: `esp-aes 256 esp-sha-hmac`
- Mode: **transport** (encrypts GRE payload, not IP header)

**Why GRE + IPsec and not just IPsec?**
- IPsec alone cannot encapsulate multicast/broadcast
- GRE creates point-to-point tunnel supporting multicast (needed for OSPF, EIGRP)
- IPsec encrypts the entire GRE tunnel transparently

### build_gre_ipsec_commands

```python
def build_gre_ipsec_commands(tunnels: list) -> list
```

Generates complete IOS commands for GRE over IPsec tunnels.

<ParamField path="tunnels" type="list[dict]">
  List of tunnel configs with keys: `tunnel_id`, `src`, `dst`, `tunnel_ip`, `tunnel_mask`, `psk`, `peer`, `map_name`, `map_seq`, `bind_iface`
</ParamField>

**Returns:** `list[str]` - IOS commands in correct order

**Generated for each tunnel:**

1. **ISAKMP Policy** (Phase 1)
   ```
   crypto isakmp policy <N>
    encr aes 256
    hash sha
    authentication pre-share
    group 2
    lifetime 86400
   ```

2. **ISAKMP Key** (PSK)
   ```
   crypto isakmp key <psk> address <peer>
   ```

3. **ACL for GRE traffic**
   ```
   ip access-list extended ACL_GRE_<tid>
    permit gre host <src> host <dst>
   ```

4. **Transform-Set** (Phase 2)
   ```
   crypto ipsec transform-set TS_GRE_<tid> esp-aes 256 esp-sha-hmac
    mode transport
   ```

5. **Crypto Map**
   ```
   crypto map <name> <seq> ipsec-isakmp
    set peer <peer>
    set transform-set TS_GRE_<tid>
    match address ACL_GRE_<tid>
   ```

6. **Bind to interface**
   ```
   interface <bind_iface>
    crypto map <name>
   ```

7. **Tunnel interface**
   ```
   interface Tunnel<tid>
    ip address <tunnel_ip> <tunnel_mask>
    tunnel source <src>
    tunnel destination <dst>
    tunnel mode gre ip
    no shutdown
   ```

<Warning>
  Compatible with **IOSvL2 15.2** experimental firmware that supports GRE + IPsec in software mode.
</Warning>

## Common Patterns

### State Management

All tab builders store widget references and data lists on the `app` object:

```python
app.sucursales = []       # List of branch dicts
app.dhcp_pools = []       # List of DHCP pool dicts
app.vlans_data = []       # List of VLAN dicts
app.static_routes = []    # List of route dicts
app.ospf_networks = []    # List of OSPF network dicts
app.qos_classes = []      # List of class-map dicts
app.pol_entries = []      # List of policy-map entries
app.service_policies = [] # List of service-policy dicts
app.gre_tunnels = []      # List of GRE tunnel dicts
```

### Listbox Update Pattern

```python
def _add_item(app):
    # 1. Validate form fields
    # 2. Append to app data list
    app.items.append(item_dict)
    # 3. Insert formatted string to listbox
    app.item_listbox.insert(tk.END, formatted_string)
    # 4. Clear form fields
```

### Protected Uplink

Multiple tabs reject operations on `UPLINK_IFACE` (Gi0/0):
- VLAN port assignment
- QoS service-policy
- GRE crypto map binding

### Enable/Disable Toggle Pattern

```python
def _toggle_feature(app):
    state = "normal" if app.chk_var.get() else "disabled"
    app.related_entry.config(state=state)
```

Used in: VLAN (DHCP/ports), Routing (OSPF)
